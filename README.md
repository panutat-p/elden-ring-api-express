## Dependencies

```shell
npm i -D typescript
npm i -D ts-node
npm i -D @types/node
npm i -D prettier
tsc --init

npm i dotenv
npm i express cors yup
npm i -D @types/express
npm i luxon
npm i -D @types/luxon
npm i uuid
npm i -D @types/uuid
npm i mongodb
```
